# Server-Base-Express-Sockets
Servidor base creado en expres con sockets para podert ser reutilizado según necesidades.

# Instrucciones

Reconstuir módulos de Node
```
npm i
```

Generar el DIST
```
tsc -w
```

Levantar servidor
```
nodemon dist/
```

* TypeScript
* Node
* Express
* cors
* socket.io
